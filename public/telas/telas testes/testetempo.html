Tutorial Interativo do Anime.js

Este é um tutorial prático para aprender a usar o Anime.js, uma biblioteca JavaScript para animações. Clique nos botões abaixo para ver diferentes tipos de animações aplicadas a um objeto simples (um quadrado vermelho) no canvas. Cada botão demonstra uma função ou propriedade específica do Anime.js. O código é comentado para facilitar o aprendizado!

Dicas: Anime.js anima propriedades de objetos JavaScript. Aqui, animamos um objeto 'quadrado' com propriedades como x, y, rotation, etc. Use 'targets' para o objeto, 'duration' para tempo, 'easing' para suavidade, e 'loop' para repetição. Veja o console para logs das animações!
